# Linux中安装Docker

最近在用docker容器，出现了 docker: command not found 的错误。

竟然发现docker都没有安装，这怎么能忍呢？

首先，导入阿里云的服务器(可自行选择)

curl -fsSL https://get.docker.com | bash -s docker --mirror Aliyun

安装成功，如下图：

开始正式安装doker

  1. 更新yum包 更新yum包管理工具库信息

yum update

被逼无奈遵循协议输入y

  1. 安装依赖软件包 安装docker环境需要的其他依赖包

yum install -y yum-utils device-mapper-persistent-data lvm2

  1. 设置yum源 设置yum工具包的源仓库

yum-config-manager --add-repo https://download.docker.com/linux/centos/docker-ce.repo

  1. 安装docker 正式安装docker程序包，这里我们选择免费使用的社区版(默认安装最新的docker稳定版本)

yum install docker-ce

  1. 启动docker服务 启动docker服务，才能使用docker命令

systemctl start docker

  1. 设置开机自启动 温馨提示：最好设置docker服务开机自启动，省去我们每次都手动启用docker的麻烦

安装完成，使用docker -v 命令查看docker版本信息：
