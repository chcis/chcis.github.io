# Spring Data Jpa概述

作为java程序员,增删改查的搬运工，再怎么也得会Spring Data。

# 什么是Spring Data？

Spring Data是Spring爹地提供的一个操作数据库的框架,其目的是统一访问SQL关系型数据库和NoSQL非关系型数据库的各种持久性存储,包括但不仅限于Mysql，Redis，MongoDB等。

# Spring Data Jpa又是什么？

  • Spring Data Jpa是SpringData框架下的一个基于JPA标准操作数据的模块。

  • SpringData Jpa 极大简化了数据库访问层代码。

  • 如何简化的呢？使用了SpringDataJpa，我们的dao层中只需要写接口，就自动具有了增删改查、分页查询等方法。

  • 当我们实现一个新的应用程序时，应该专注于业务逻辑，而不是技术复杂性和样板代码，这也是Spring Data Jpa和Java Persistent API(JPA)规范非常受欢迎的原因。

# 学习中遇到的困难

控制台报错：java.lang.NoClassDefFoundError: javax/xml/bind/JAXBException

解析：

JAXB API被认为是Java EE API，因此在Java SE 9中不再包含在默认的classpath上。在Java 11中，它们完全被JDK移除。

解决方案：

pom文件中添加下面两个依赖即可解决版本问题

<!-- API, java.xml.bind module -->
<dependency>
    <groupId>jakarta.xml.bind</groupId>
    <artifactId>jakarta.xml.bind-api</artifactId>
    <version>2.3.2</version>
</dependency>
<!-- Runtime, com.sun.xml.bind module -->
<dependency>
    <groupId>org.glassfish.jaxb</groupId>
    <artifactId>jaxb-runtime</artifactId>
    <version>2.3.2</version>
</dependency>
