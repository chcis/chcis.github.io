# RabbitMQ与SpringBoot整合

上一篇博客我写了RabbitMQ的环境搭建和RabbitMQ的六种工作模式详解，玩起来是相当的复杂和费劲。

作为程序员来说，研究底层原理和底层方法很爽很酷(我好喜欢)，但也要有一套简便的方法去实现需求。

SpringBoot集成RabbitMQ非常简单，简单的使用配置非常少，springboot提供了spring-boot-starter-amqp项目对消息的各种支持。
